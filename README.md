#Fitness Tracker (Flutter + Python Backend)

![Flutter](https://img.shields.io/badge/Flutter-Mobile-blue)
![Python](https://img.shields.io/badge/Python-Backend-yellow)
![Status](https://img.shields.io/badge/Status-Active-success)

A cross-platform fitness tracking application built using **Flutter** with a **Python-based backend** for data handling and analytics.

---

# Features

Track daily workouts and activity
Calorie tracking and basic analytics
Clean, responsive Flutter UI
Backend integration with Python APIs
Data persistence and retrieval

---

# Tech Stack

- **Frontend:** Flutter (Dart)
- **Backend:** Python (Flask)
- **Database:** (MySQL / MongoDB - configurable)

---

# How to Run

# 1. Clone the repository
```bash
git clone https://github.com/nimeghala45/fitness-tracker-flutter.git
cd fitness-tracker-flutter

Run flutter app:
flutter pub get
flutter run

start backend:
cd backend
pip install -r requirements.txt
python app.py
